# User Profile Page (React + Props)

This is a minimal Create React App project that renders a user profile page using props.

## Quick Start

```bash
# 1) Install dependencies
npm install

# 2) Start the dev server
npm start
```

## Files of Interest

- `src/userData.js` – hardcoded user object
- `src/ProfileCard.js` – basic user details via props
- `src/ProfileBio.js` – bio via props
- `src/UserProfile.js` – composes the profile components
- `src/App.js` – mounts `UserProfile`

You can change the user data in `src/userData.js` and see it reflected immediately.
